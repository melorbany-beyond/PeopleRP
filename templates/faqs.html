{{ _('What do the different project statuses mean?') }}

{{ _('Projects can have the following statuses:') }}

  • {{ _('Active: The project is currently in progress') }}
  • {{ _('On Hold: The project is temporarily paused') }}
  • {{ _('Completed: The project has been finished') }}
  • {{ _('Cancelled: The project has been terminated') }}
  • {{ _('Not Started: The project start date is in the future') }}
  • {{ _('Overdue: The project has passed its end date without being completed') }}

{{ _('Why do I see two different statuses for my project?') }}

{{ _('The system shows both an automated status (based on dates) and your manual status. The automated status helps track timeline progress, while the manual status reflects the actual project state you\'ve set.') }}

{{ _('Resource Allocation') }}

{{ _('How is total allocation calculated?') }}

{{ _('Total allocation is calculated based on project status:') }}

  • {{ _('Not Started projects: 0%% allocation (not counted)') }}
  • {{ _('Active projects: 100%% of assigned allocation') }}
  • {{ _('On Hold projects: 100%% of assigned allocation') }}
  • {{ _('Overdue projects: 100%% of assigned allocation') }}
  • {{ _('Completed projects: 0%% allocation (not counted)') }}

{{ _('For example, if someone is assigned 30%% to an Active project and 40%% to a Not Started project, their total allocation would be 30%% since the Not Started project is not counted.') }}

{{ _('What do the allocation colors mean?') }}

  • {{ _('Green: Less than 50%% allocation - Available for more work') }}
  • {{ _('Yellow: 50-80%% allocation - Moderately loaded') }}
  • {{ _('Red: Over 80%% allocation - Heavily loaded') }}

{{ _('Team Management') }}

{{ _('How do I assign someone to multiple projects?') }}

{{ _('You can assign a team member to multiple projects as long as their total allocation across active projects doesn\'t exceed 100%%. Go to each project\'s assignments page and add them with the appropriate allocation percentage.') }}

{{ _('What happens to team assignments when a project is completed?') }}

{{ _('When a project is marked as completed:') }}

  • {{ _('The assignments are preserved for historical record') }}
  • {{ _('The allocations no longer count toward team members\' total allocation') }}
  • {{ _('Team members become available for new assignments') }}
